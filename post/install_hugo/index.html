Hugo环境搭建——Windows环境下

Sep 24, 2019
1 min read

★内容参考B站Up主CodeSheep，视频地址：https://www.bilibili.com/video/av51574688 我整理了Windows下的操作方法，如下：

1：在https://github.com/gohugoio/hugo/releases地址下找到Windows压缩文件下载，你可以下载32或者64位的

2:如何看hugo安装成没成功，打开cmd，输入hugo version，不提示错误，输出版本号，则代表安装成功

3：安装成功后，用Hugo来创建自己的博客，命令行:hugo new site myblog 注：myblog为文件夹名字

4:我们进入到myblog里面

5:下载设置主题，去https://themes.gohugo.io/选择自己喜欢的主题进行下载

6：这里我选择m10c，以他为例子。注：前提一定要下git并且配置好！

7：启动博客命令，输入 hugo server -t m10c –buildDrafts

8：接下来就是创建文章，我们继续在根目录下创建

9：那怎么写呢，推荐用VS Code来写，写完后记得保存

10：然后再根目录下输入指令 hugo server -t m10c –buildDrafts

11：再打开http://localhost:1313

12：把博客部署的远端github上，首先创建仓库

13：仓库已经有了，但是个空仓库，接下来敲命令hugo –theme=m10c –baseUrl=“地址” –buildDrafts

14：成功后会在根目录下生成一个public文件

15：把这个public传到github仓库上，首先进入到public

16：要把本地public和远程github进行关联

17：然后还有个push命令，git push -u origin master，期间会让你输入账号密码，输入即可

18：最后，我们拿仓库地址访问下
